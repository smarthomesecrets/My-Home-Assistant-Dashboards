# My Home Assistant Dashboards

A collection of my personal Home Assistant dashboard YAML files, free to download and adapt for your own setup.

Each dashboard lives in its own folder. Open the folder for the dashboard you want and read the README inside before doing anything else — it lists the required HACS plugins, every placeholder you need to replace, and instructions for adding or removing entities.

---

## Dashboards

| Folder | Dashboard | Description |
|---|---|---|
| `tab-kitchen/` | Tab Kitchen | Kitchen tablet dashboard with 7 views: photos, home overview, kids chores, media control, sports scores, HVAC, and live cameras |
| `daily-messages/` | Daily Messages | Per-user morning briefing with weather, calendars, energy summary, and AI-generated daily content |

---

## How to Use These Files

1. Pick a dashboard folder
2. Read the `README.md` inside it
3. Use **Find & Replace** to swap all `ALL_CAPS` placeholders for your own entity IDs
4. Paste the view YAML into your Home Assistant dashboard via **Edit Dashboard → Raw Configuration Editor**
